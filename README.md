# Festiva Papelaria Personalizada

Site simples e completo para vendedores de papelaria personalizada de festa.

## Funcionalidades
- Registro e login na tela inicial.
- Dashboard com áreas: **Geral**, **Produtos**, **Pedidos**, **Vendas**.
- Cadastro e exclusão de produtos com confirmação por senha.
- Tempo/data de criação do produto.
- Página pública da loja por usuário (`?loja=usuario`).
- Compra com formulário completo do cliente (tema, gênero, idade, observações, contato).
- Geração de número de pedido.
- Exibição de contato do vendedor após envio.
- Pagamento via PIX (QR code + copiar código).
- Upload e pré-validação simples de comprovante.
- Lista de pedidos para vendedor com busca por número.
- Área de vendas com total vendido e gráfico por produto.

## Como usar
1. Abra o arquivo `index.html` no navegador.
2. Crie uma conta e faça login.
3. Cadastre produtos no painel.
4. Compartilhe o link da loja pública mostrado na aba **Geral**.

## Observação técnica
Este projeto usa `localStorage` para persistência local (sem backend).
